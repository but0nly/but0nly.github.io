| 步骤                         | 做了什么                                                                                                    |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `DNS`解析                    | 拿到域名解析后的`IP`地址，请求`IP`地址，这期间会进行`TCP`连接（三次握手）                                   |
| 获取到`HTML`资源             | 服务端给我们发送`HTML`资源，这时候的`HTML`还是字符串                                                        |
| 解析`HTML`源码               | 浏览器开始解析`HTML`源码                                                                                    |
| 请求静态资源                 | 解析过程中发现静态资源就会去请求资源。例如`js`、`css`、图片、视频等等，如果是强缓存就不会请求，直接从本地拿 |
| 把`HTML`文本转换成结构化数据 | `HTML` 构建 `DOM`树，`CSS` 构建 `CSSOM` 树，两者结合形成`Render Tree`                                       |
| `Render Tree`绘制到页面      | 计算各个`DOM`的尺寸、定位，最后根据`位置`信息填充像素到页面                                                 |
| 遇到`JS`可能会执行           | `defer`、`async`                                                                                            |
