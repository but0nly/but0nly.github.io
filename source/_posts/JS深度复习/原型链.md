### 定义
**原型**本质个*对象*

- *显式原型*
	- 构造函数的原型：`Construct.prototype`
		- ```js
		  function Cat(name){
			  this.name = name
		  }
		  
		  // `Construct.prototype` 本质是个对象
		  // 作用： 用于存储所有由构造函数创建的实例所共享的属性和方法。
		  Cat.prototype.sayName = function(){
			  console.log(this.name)
		  }
		  
		  
		  const kitty = new Cat('kitty');
		  kitty.sayName(); // kitty
		  ```
- *隐式原型*
	- 实例的`__proto__`
	- 是构成**原型链**的链接
	- *题外知识*：`__proto__`是非标准且已弃用的属性。推荐使用`Object.getPrototypeOf()`来获取对象原型。


*实例的原型 === 构造函数的原型*, 示例：
```js
const obj = new Object();


obj.__proto__ ==== Object.prototype // true 
```



### 基于原型的执行规则

1. 获取一个对象的属性或者方法时
2. 先会在对象自身去查找
3. 如果没有，就会顺着属性`__proto__`去查找，直到找到属性、对象或着达原型链的终点（`null`）
